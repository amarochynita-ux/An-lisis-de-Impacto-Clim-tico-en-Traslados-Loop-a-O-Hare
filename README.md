# 🚕 Análisis de Impacto Climático en Traslados: Loop a O'Hare

## 📌 Resumen del Proyecto
Este proyecto aplica **estadística inferencial con Python** para determinar si las condiciones climáticas son un factor determinante en la duración de los viajes urbanos en Chicago. El análisis se centra en la ruta estratégica hacia el Aeropuerto Internacional O'Hare, proporcionando una base científica para la toma de decisiones operativas.

## 📊 Marco Estadístico (Prueba de Hipótesis)
Para validar el impacto del clima, se implementó una **Prueba t de muestras independientes**:
*   **H0 (Nula):** La lluvia no genera una diferencia significativa en los tiempos de traslado.
*   **H1 (Alternativa):** La lluvia impacta significativamente en la duración promedio de los viajes.
*   **Metodología:** Comparación de medias entre grupos independientes para descartar el azar y confirmar la causalidad climática.

## 🏢 Beneficios Estratégicos para la Empresa
El valor real de este análisis se traduce en las siguientes ventajas competitivas:

1.  **Optimización de la Satisfacción del Cliente (NPS):** Permite ajustar los algoritmos de **ETA (Tiempo Estimado de Llegada)**, reduciendo la frustración del usuario mediante expectativas de tiempo realistas en días de lluvia.
2.  **Eficiencia en Precios Dinámicos:** Proporciona sustento técnico para implementar tarifas dinámicas por clima, protegiendo el margen de ganancia y compensando el esfuerzo adicional del conductor.
3.  **Gestión Inteligente de Flota:** Mejora la predicción de disponibilidad de unidades; al saber que los viajes durarán más, la empresa puede redistribuir vehículos para equilibrar la oferta y demanda.
4.  **Reducción de Costos Operativos:** Disminuye la tasa de cancelaciones y reclamos por demoras, optimizando la carga de trabajo en el área de soporte al cliente.

## 🚀 Conclusiones
Este estudio demuestra que el clima es un **factor de riesgo crítico**. Pasar de una operación reactiva a una **operación predictiva** basada en datos permite a la empresa mantener su competitividad y confiabilidad incluso en condiciones adversas.
